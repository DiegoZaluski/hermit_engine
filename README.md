## Overview
Hermit is a lightweight runtime designed to load and execute GGUF (GPT-Generated Unified Format) models using the Candle ML framework. The project focuses on simplicity and performance, starting with a minimal implementation that can be extended with optimizations and features over time.

## Key Features
- **Minimal Architecture**: Clean, modular design for easy maintenance
- **GGUF Support**: Native GGUF file format parsing and loading
- **Candle Integration**: Built on Rust's Candle ML framework
- **Performance Focused**: Optimized for efficient inference
- **Extensible**: Designed for future enhancements and optimizations
